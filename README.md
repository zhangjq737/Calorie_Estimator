# Calorie_Estimator
Suggested steps to run this prototype:
## Step 1. Create virtual environment
create virtual environment using python 3.13.9:

`$ conda create --name calorie_estimator python=3.13.9`

## Step 2. Activate virtual environment:
Navigate to project folder then:

`$ conda activate calorie_estimator`

## Step 3. Install dependencies
`$ pip install -r requirements.txt`

## Step 4. Configure .env
create .env file in root folder and create two variables:

>USDA_API_KEY = "YOUR_USDA_API_KEY"

>USDA_BASE_URL = "https://api.nal.usda.gov/fdc/v1"`

Done!
